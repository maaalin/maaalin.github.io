Web design

What: Assignment in TNGD46 Document Production and Publishing at Linköping University

Team: Worked together with Sofia Krantz. Project didn't include any designated roles

Client: Bring Me The Horizon (fictional case)

Target group: Fans of the band Bring Me The Horizon

Objective: To code a functional website using HTML, CSS and JavaScript

Challenges: Two main technical challenges: incorporating a hover effect on the discography page, and making it possible to change the colour theme of the website and not have the change reset when the user navigates to a different page. We managed to get both of these functions to work through a lot of research (shout-out to Stack Overflow) and tons of trial and error.
